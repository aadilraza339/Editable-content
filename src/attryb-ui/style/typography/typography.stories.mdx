# Display
Display is for titling various interface elements, such as empty states and modals.

**Default Font:** Inter, sans-serif

**Applicable Weights:** 400 (regular), 500 (medium), 700 (bold)

## Display x-large

**CSS ClassName:** `.display-xl`

**Font Size:** 36px / 3rem

**Line Height:** 40px / 3.25rem

**Default Weight:** 500

# Headings
Group of heading specific typographies

**Default Font:** Inter, sans-serif

**Applicable Weights:** 400 (regular), 500 (medium), 700 (bold)

## Page Heading

PageHeading is reserved for the title of a screen.

**CSS ClassName:** `.page-heading`

**Font Size:** 20px / 1.67rem

**Line Height:** 24px / 2rem

**Default Weight:** 700

<Typeset
    fontSizes={[
        Number(typography.size.ph)
    ]}
    fontWeight={typography.weight.bold}
    sampleText={SampleText}
    fontFamily={typography.type.primary}
/>

## Heading

Heading should always be used for titles of top-level sections of a screen. If the sections of a screen are represented by cards, each cardâ€™s title should use the Heading style.

**CSS ClassName:** `.heading`

**Font Size:** 17px / 1.42rem

**Line Height:** 20px / 1.67rem

**Default Weight:** 500

<Typeset
    fontSizes={[
        Number(typography.size.h)
    ]}
    fontWeight={typography.weight.medium}
    sampleText={SampleText}
    fontFamily={typography.type.primary}
/>

## Sub Heading

If a top-level section of a screen has subsections, use the Subheading style for titling those subsections. Subheading should never appear as the first element in a card. Only use with titles (vs. sections of content).

**CSS ClassName:** `.subheading`

**Font Size:** 14px / 1.17rem

**Line Height:** 16px / 1.33rem

**Default Weight:** 500

<Typeset
    fontSizes={[
        Number(typography.size.su)
    ]}
    fontWeight={typography.weight.medium}
    sampleText={SampleText}
    fontFamily={typography.type.primary}
/>
